# CJK Bottom Margin Auto Fix

**CJK Bottom Margin Auto Fix** is an Adobe InDesign UXP plugin designed for precise CJK body text layout.

It calculates the ideal bottom margin from a selected primary text frame and applies the result to the linked master page. It can also generate a baseline grid based on the top margin and leading.

## Features

- Calculates the bottom margin from a selected primary text frame
- Displays page height, top margin, font size, leading, line count, and calculated bottom margin
- Applies the calculated value to the linked master page
- Adjusts the primary text frame height
- Optionally creates a baseline grid based on the top margin and leading
- Supports Korean, Japanese, Simplified Chinese, and Traditional Chinese interface languages
- Supports light and dark UI modes

## Intended Use

This plugin is intended for CJK typography and book layout workflows, especially Korean, Japanese, Simplified Chinese, and Traditional Chinese body text.

It is not designed as a general-purpose Latin text layout tool, because Latin typography may require font-specific descender and baseline considerations.

## How to Use

1. Open an Adobe InDesign document.
2. Select a primary text frame that contains text.
3. Check the calculated layout information shown in the plugin panel.
4. Enable or disable baseline grid generation as needed.
5. Click **Apply**.

The plugin applies the calculated bottom margin to the linked master page and adjusts the primary text frame height.

## Supported Languages

- Korean
- Japanese
- Simplified Chinese
- Traditional Chinese
