## 2. Intended Use

CJK Bottom Margin Auto Fix is designed for CJK typography and book layout workflows in Adobe InDesign.

The plugin calculates and applies bottom margin values from a selected primary text frame. It may also generate a baseline grid based on the top margin and leading.

The plugin is intended for Korean, Japanese, Simplified Chinese, and Traditional Chinese body text layout. It is not designed as a general-purpose Latin text layout tool.
